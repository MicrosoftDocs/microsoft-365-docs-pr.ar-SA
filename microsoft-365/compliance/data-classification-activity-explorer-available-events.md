# <a name="labeling-activities-that-are-available-in-activity-explorer"></a>أنشطة التسمية المتوفرة في مستكشف النشاط

## <a name="sensitivity-label-applied"></a>تم تطبيق وصف الحساسية

يتم إنشاء هذا الحدث في كل مرة يتم فيها تسمية مستند غير مسمى أو يتم إرسال بريد إلكتروني مع وصف الحساسية.

- يتم التقاطها في وقت الحفظ في تطبيقات Office الأصلية وتطبيقات الويب.
- يتم التقاطها في وقت التكرار لعميل التسمية الموحد ل Azure حماية البيانات (AIP).
- يمكن أيضا مراقبة إجراءات تسميات الترقية والأدنى عبر حقل *نوع حدث التسمية* وعامل التصفية.

|مصدر  |تم الإبلاغ عنها في مستكشف النشاط | ملاحظه  |
|---------|---------|---------|
| Word وExcel وPowerPoint|نعم |
|Outlook| نعم | |
|SharePoint online، OneDrive|نعم | |
|Exchange        |نعم         | |
|عميل موحد ل Azure حماية البيانات (AIP) وماسح ضوئي موحد ل AIP |نعم |تم تعيين إجراء *التسمية الجديد* ل AIP إلى *التسمية المطبقة* في مستكشف النشاط   |
|حماية البيانات في Microsoft (MIP) SDK         |نعم|تم تعيين إجراء *التسمية الجديد* ل AIP إلى *التسمية المطبقة* في مستكشف النشاط|
|خدمة إدارة الحقوق (RMS)         |غير قابل للتطبيق         | |
|Power BI لسطح المكتب والويب        | لا| يمكن الوصول إليه في سجلات تدقيق Microsoft 365         |
|Microsoft Defender for Cloud Apps         |لا|         |

## <a name="sensitivity-label-changed"></a>تم تغيير تسمية الحساسية

يتم إنشاء هذا الحدث في كل مرة يتم فيها تحديث وصف الحساسية على المستند أو البريد الإلكتروني.

- بالنسبة إلى العميل الموحد ل AIP، تم تغيير الماسح الضوئي الموحد ل AIP ومصادر MIP SDK *وتسمية ترقية* AIP وخرائط إجراء *التسمية* إلى *تسمية* مستكشف النشاط

- يتم التقاطها عند نقطة الحفظ في تطبيقات Office الأصلية وتطبيقات الويب.
- يتم التقاطها في وقت التكرار لتطبيقات عميل التسمية الموحدة ل AIP والماسح الضوئي
- يمكن أيضا مراقبة إجراءات تسميات الترقية والأدنى عبر حقل *نوع حدث التسمية* وعامل التصفية. يتم أيضا التقاط نص *الضبط* باستثناء SharePoint Online وOneDrive.
- يجمع وصف الحساسية الذي تم إجراؤه في تطبيقات Office الأصلية على Outlook الإجراء الأخير الذي تم إنشاؤه قبل إجراءات حفظ الملفات/إرسال البريد الإلكتروني. على سبيل المثال، إذا قام المستخدم بتغيير التسمية على رسالة بريد إلكتروني عدة مرات قبل الإرسال، يتم تسجيل التسمية الأخيرة التي تم العثور عليها على البريد الإلكتروني عند إرسالها في سجل التدقيق ثم يتم الإبلاغ عنها في مستكشف النشاط.

|Outlook         |نعم         |
|SharePoint Online، OneDrive         |نعم         |
|Exchange         |نعم         |
|عميل AIP الموحد         |نعم         |

|Microsoft Defender for Cloud Apps     |لا         |         |

## <a name="known-issues"></a>المشاكل المعروفة
  
- عند عرض تلميح أداة التسمية الموصى بها لمستخدم نهائي، لا يتم التقاطه. ولكن إذا اختار المستخدم تطبيق التسمية الموصى بها، فسيتم عرض التسمية ضمن الحقل *"كيفية التطبيق* " كما *هو موصى به*.

- نص الضبط غير متوفر حاليا عند تخفيض وصف الحساسية من SharePoint وOneDrive.

- أنواع المعلومات الحساسة غير متوفرة حاليا لأنشطة التسمية التلقائية من Word وExcel وPowerPoint وOutlook، بالإضافة إلى SharePoint Online وOneDrive.
