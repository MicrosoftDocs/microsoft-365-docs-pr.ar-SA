# <a name="microsoft-365-compliance-center"></a>مركز التوافق في Microsoft 365

إذا كنت مهتما بوضعية التوافق في مؤسستك، ستحب <a href="https://go.microsoft.com/fwlink/p/?linkid=2077149" target="_blank">مركز التوافق في Microsoft 365.</a> يوفر مركز التوافق في Microsoft 365 الوصول السهل إلى البيانات والأدوات التي تحتاج إلى إدارتها لتلبية احتياجات التوافق في مؤسستك.

اقرأ هذه المقالة للتعرف على مركز التوافق في Microsoft 365 وكيفية الحصول عليها والأسئلة التي يتم [](#how-do-i-get-the-compliance-center)طرحها بشكل متكرر [والخطوات التالية](#next-steps). [](#frequently-asked-questions)

[![مركز التوافق في Microsoft 365 الصفحة الرئيسية.](../media/m365-compliance-center-home.png)](https://compliance.microsoft.com)

## <a name="welcome-to-microsoft-365-compliance"></a>مرحبا بك في Microsoft 365 التوافق

عندما تذهب إلى مركز التوافق في Microsoft 365 للمرة الأولى، يتم الترحيب بك برسالة الترحيب التالية:

![مركز التوافق في Microsoft 365 مقدمة.](../media/m365-compliance-center-welcome-steps.png)

يوفر لك شعار الترحيب بعض اللافتات حول كيفية البدء، مع الخطوات التالية، ودعوة لك لتعطينا ملاحظاتك.

## <a name="card-section"></a>مقطع البطاقة

عند زيارة صفحة مركز التوافق في Microsoft 365 لأول مرة، يعرض لك قسم البطاقة على الصفحة الرئيسية نظرة سريعة حول كيفية عمل مؤسستك مع توافق البيانات والحلول المتوفرة لمنظمتك وملخص لأي تنبيهات نشطة.

من هنا، يمكنك:

- راجع بطاقة **إدارة التوافق من Microsoft** ، التي ستقودك إلى حل [إدارة](compliance-manager.md) التوافق. تساعد إدارة التوافق على تبسيط الطريقة التي تدير بها التوافق. تحسب هذه العملية درجة تستند إلى المخاطر تقيس تقدمك نحو إكمال الإجراءات الموصى بها التي تساعد على تقليل المخاطر المتعلقة بحماية البيانات والمعايير التنظيمية. كما يوفر قدرات سير العمل ورسم خريطة مضمنة للتحكم لمساعدتك على تنفيذ إجراءات التحسين بفعالية.

    ![بطاقة إدارة التوافق مركز التوافق في Microsoft 365.](../media/m365-compliance-center-compliance-manager-card.png)

- راجع بطاقة **كتالوج Solution** الجديدة، التي تربط إلى مجموعات من [](microsoft-365-solution-catalog.md) الحلول المتكاملة التي يمكنك استخدامها لمساعدتك على إدارة سيناريوهات التوافق من النهاية إلى النهاية. قد تتضمن إمكانيات الحل وأدواته مجموعة من السياسات والتنبيهات والتقارير والمزيد.

    ![بطاقة كتالوج مركز التوافق في Microsoft 365.](../media/m365-compliance-center-solution-catalog-card.png)

- راجع **بطاقة التنبيهات** النشطة، التي تتضمن ملخصا للتنبيهات الأكثر [](alert-policies.md) نشاطا وتتضمن ارتباط حيث يمكنك عرض معلومات أكثر تفصيلا، مثل الخطورة، الحالة، الفئة، والمزيد.

    ![بطاقة التنبيهات النشطة مركز التوافق في Microsoft 365.](../media/m365-compliance-center-active-alerts-card.png)

يمكنك أيضا استخدام ميزة إضافة  بطاقات لإضافة بطاقات إضافية، مثل واحدة تعرض توافق تطبيقات السحابة في مؤسستك، وأخرى تعرض بيانات حول المستخدمين الذين لديهم ملفات مشتركة، مع ارتباطات إلى [Defender for Cloud Apps](/cloud-app-security/) أو أدوات أخرى حيث يمكنك استكشاف البيانات.

![تفاصيل بطاقة مركز التوافق الإضافية.](../media/m365-compliance-center-additional-cards.png)

## <a name="how-do-i-get-the-compliance-center"></a>كيف يمكنني الحصول على مركز التوافق؟

- إذا لم يكن لديك مركز التوافق في Microsoft 365 الجديد بالفعل، فسوف تحصل عليه قريبا. تتوفر مركز التوافق في Microsoft 365 بشكل عام الآن Microsoft 365 SKU.
- لزيارة مركز التوافق في Microsoft 365، كمسؤول عام أو مسؤول التوافق أو مسؤول بيانات التوافق[https://compliance.microsoft.com](https://compliance.microsoft.com)، انتقل إلى سجل الدخول.

## <a name="frequently-asked-questions"></a>الأسئلة المتكررة

**لماذا لا يمكنني رؤية مركز التوافق في Microsoft 365 بعد؟**

أولا، تأكد من أن لديك الأذونات والتراخيص المناسبة. بعد ذلك، <a href="https://go.microsoft.com/fwlink/p/?linkid=2077149" target="_blank">سجل الدخول إلى</a> مركز التوافق في Microsoft 365. إذا لم يتم رؤية مركز التوافق الجديد بعد، فسوف تحصل عليه قريبا.

![موارد إضافية.](../media/m365-compliance-center-more-resources.png)

## <a name="next-steps"></a>الخطوات التالية

- **تفضل بزيارة Microsoft Compliance Manager** لمشاهدة نقاط التوافق وبدء إدارة التوافق لمنظمتك. لمعرفة المزيد، راجع [إدارة التوافق](compliance-manager.md).

- **قم بتكوين سياسات إدارة مخاطر insider** للمساعدة في تقليل المخاطر الداخلية وتمكينك من الكشف عن الأنشطة ذات المخاطر في مؤسستك وتحريها واتخاذ إجراء بشأنها. راجع [إدارة مخاطر Insider](insider-risk-management.md).

- **راجع سياسات منع** فقدان البيانات في مؤسستك وأ إجراء التغييرات المطلوبة كما تقتضي الحاجة. لمعرفة المزيد، راجع [التعرف على منع فقدان البيانات](dlp-learn-about-dlp.md).

- **تعرف على Microsoft Defender for Cloud Apps و قم بإعداده**. راجع [التشغيل السريع: بدء استخدام Microsoft Defender لتطبيقات السحابة](/cloud-app-security/getting-started-with-cloud-app-security).

- **تعرف على نهج توافق الاتصالات** وأنشئها لتحديد الانتهاكات المتعلقة ونهج قواعد السلوك الخاصة بالشركة بسرعة. راجع [توافق الاتصالات في Microsoft 365](communication-compliance.md).

- **تفضل بزيارة مركز التوافق في Microsoft 365،** وتأكد من مراجعة أي تنبيهات أو مخاطر محتملة تنشأ. انتقل إلى [https://compliance.microsoft.com](https://compliance.microsoft.com) ثم سجل الدخول.
