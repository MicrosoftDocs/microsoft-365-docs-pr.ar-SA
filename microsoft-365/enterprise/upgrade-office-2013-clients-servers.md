# <a name="resources-to-help-you-upgrade-from-office-2013-clients-and-servers"></a>موارد لمساعدتك على الترقية من عملاء Office 2013 وخوادمه

إذا كنت تستخدم عملاء Office 2013 وخوادمه، فكن على علم بأن ***الدعم ينتهي في 11 أبريل 2023***. إذا لم تكن قد بدأت بالفعل في الترقية منها إلى إصدارات أحدث، نوصي بالبدء الآن.

أثناء التخطيط للترقية، فكر في الانتقال إلى Microsoft 365.

- يحتوي Microsoft 365 على خدمات مستندة إلى السحابة، مثل Exchange Online وSharePoint وTeams.

- يوفر Microsoft 365 Apps for enterprise تطبيقات عميل Office التي يمكنك تثبيتها على الأجهزة المحلية. يتم تحديث التطبيقات بانتظام من سحابة Microsoft مع ميزات إنتاجية وأمان جديدة.

|---|---|
|Office 2013 (تطبيقات سطح المكتب)<br/>Project 2013<br/>Visio 2013|[تخطيط ترقية من الإصدارات القديمة من Office إلى Microsoft 365 Apps](/deployoffice/endofsupport/plan-upgrade-older-versions-office.md)|
|Exchange Server 2013|[مخطط نهاية الدعم ل Exchange 2013](exchange-2013-end-of-support.md)|
|SharePoint Server 2013 <br/> SharePoint Foundation 2013|[الترقية من SharePoint 2013](upgrade-from-sharepoint-2013.md)|
|Project Server 2013|[مخطط نهاية الدعم ل Project Server 2013](project-server-2013-end-of-support.md)|
|Lync Server 2013|[الترقية من Lync Server 2013](upgrade-from-lync-2013.md)|
