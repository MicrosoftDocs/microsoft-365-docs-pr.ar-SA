# <a name="test-microsoft-365-with-test-lab-guides-tlgs"></a>اختبار Microsoft 365 باستخدام Test Lab Guides (TLGs)

تساعدك مجموعات TLG على التعرف بسرعة على منتجات Microsoft. إنها رائعة للحالات التي تحتاج فيها إلى تقييم تقنية أو تكوين قبل أن تقرر ما إذا كان ذلك مناسبا لك وقبل البدء في التصميم والتخطيط والنشر للمستخدمين. تساعدك التجربة العملية "لقد بنيتها بنفسي وتعمل" على فهم متطلبات النشر لمنتج أو حل جديد حتى تتمكن من التخطيط بشكل أفضل لاستضافته في الإنتاج.
  
تنشئ مجموعات TLGs أيضا بيئات تمثيلية لتطوير التطبيقات واختبارها، والمعروفة أيضا باسم بيئات التطوير/الاختبار.
  
![اختبار دلائل المعمل في سحابة Microsoft.](../media/24ad0d1b-3274-40fb-972a-b8188b7268d1.png)
  
## <a name="microsoft-365-devtest-environment"></a>بيئة التطوير/الاختبار Microsoft 365

استخدم هذه المقالات لإنشاء بيئة التطوير/الاختبار Microsoft 365:
  
- [التكوين الأساسي الخفيف](lightweight-base-configuration-microsoft-365-enterprise.md)
    
    إنشاء اشتراك تجريبي Microsoft 365 Enterprise E5.

- [التكوين الأساسي لمحاكاة المؤسسة](simulated-ent-base-configuration-microsoft-365-enterprise.md)
    
    إنشاء إنترانت مبسط يعمل في خدمات البنية الأساسية ل Microsoft Azure واشتراك تجريبي Microsoft 365 Enterprise E5. 

    هذا اختياري ومطلوب إذا كنت تريد إنشاء تكوين مؤسسة محاكاة لهوية مختلطة.
    
للحصول على مجموعات TLG إضافية تنطبق على كل من Office 365 و Microsoft 365، انتقل إلى [Test Lab Guides](m365-enterprise-test-lab-guides.md).
