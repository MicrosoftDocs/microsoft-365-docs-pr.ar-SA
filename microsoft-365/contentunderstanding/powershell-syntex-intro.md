# <a name="manage-sharepoint-syntex-with-powershell"></a>إدارة SharePoint Syntex باستخدام PowerShell

> [!IMPORTANT]
> إن SharePoint Syntex PowerShell cmdlets وجميع مكونات PnP الأخرى هي أدوات مفتوحة المصدر يدعمها مجتمع نشط يوفر الدعم لها. لا يوجد دعم SLA لأداة المصدر المفتوح من قنوات دعم Microsoft الرسمية.

بالنسبة إلى هذه السيناريوهات، يمكن استخدام SharePoint Syntex cmdlets في الوحدة النمطية PnP PowerShell للتفاعل مع النماذج والتفسيرات. لمعرفة المزيد حول كيفية تثبيت هذه الوحدة النمطية، راجع [نظرة عامة حول PnP PowerShell](/powershell/sharepoint/sharepoint-pnp/sharepoint-pnp-cmdlets).

حدد من السيناريوهات التالية للتعرف على كيفية استخدام PowerShell لإدارة SharePoint Syntex:

- [**نماذج الاستيراد والتصدير**](powershell-syntex-import-export.md)

    ابدأ هنا إذا كنت تريد التعرف على كيفية تصدير نماذج فهم المستندات من مركز محتوى، والاستيراد إلى مركز محتوى أو مستأجر آخر.

- [**النشر**](powershell-syntex-publishing.md)

    ابدأ هنا إذا كنت تريد التعرف على كيفية استخدام PowerShell لنشر نماذج فهم المستندات لمكتبات المستندات.

- [**المعالجة**](powershell-syntex-processing.md)

    ابدأ هنا إذا كنت تريد التعرف على كيفية استخدام PowerShell لبدء معالجة فهم المستندات على مكتبة مستندات.

- [**تفسيرات**](powershell-syntex-explanations.md)

    ابدأ هنا إذا كنت تريد التعرف على كيفية إنشاء قوالب شرح مخصصة لفهم المستند باستخدام PowerShell.
