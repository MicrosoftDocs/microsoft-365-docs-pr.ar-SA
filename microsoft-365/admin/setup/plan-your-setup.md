# <a name="plan-your-setup-of-microsoft-365-for-business"></a>التخطيط لإعداد Microsoft 365 للأعمال

هذه المقالة للأشخاص الذين اشتركوا في خطة Microsoft 365 للأعمال.
  
قبل نقل مؤسستك إلى Microsoft 365، هناك متطلبات تحتاج إلى تلبيتها، ومعلومات تحتاج إليها في متناول يدك، والقرارات التي يجب اتخاذها.

## <a name="overview-of-microsoft-365-business-premium-setup"></a>نظرة عامة Microsoft 365 Business Premium الإعداد

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE4jZwg?autoplay=false]

تهانينا على قرارك بنقل عملك إلى السحابة باستخدام Microsoft 365! سواء كان لديك شخص واحد في شركتك أو 20 عاما، فإن القيام ببعض التخطيط سيساعدك على الحصول على أفضل Microsoft 365 Business Premium.

## <a name="info-to-have-on-hand-before-you-run-the-setup-wizard"></a>المعلومات التي يجب أن تكون في متناول اليد قبل تشغيل معالج الإعداد

عندما تكون جاهزا لتشغيل معالج الإعداد وتحريك مجالك إلى Microsoft 365، إليك المعلومات التي ستحتاج إليها:
  
- قائمة الأشخاص الذين تريد إضافتهم إلى Microsoft 365. حتى لو قمت بالفعل إضافتها إلى Microsoft 365، إذا كنت تقوم بتحديث معلومات المجال، يجب إدخال أسمائها هنا.

- كيف ستعمل على إعلام الموظفين لديك بشعار المستخدم وكلمة المرور حتى يمكنهم تسجيل الدخول. هل ستتصل بهم باستخدام المعلومات؟ أو أرسله إلى عنوان بريده الإلكتروني الشخصي؟ لن يكون لديهم حق الوصول إلى بريدهم الإلكتروني، لذا لا يمكنك استخدام ذلك.

- إذا كان لديك اسم مجال لمنظمتك (مثل **contoso.com)** وخططت لاستخدام البريد الإلكتروني من Microsoft، يجب أن تعرف مكان تسجيل مجالك وأن لديك معلومات تسجيل الدخول.

## <a name="what-happens-when-you-run-the-microsoft-365-setup-wizard"></a>ماذا يحدث عند تشغيل معالج Microsoft 365 الإعداد

يوصلك معالج الإعداد عبر تثبيت تطبيقات Microsoft 365 على الكمبيوتر، وإضافة مجالك والتحقق منه، وإضافة المستخدمين وتعيين التراخيص لهم، وربط مجالك.

> [!NOTE]
> إذا كنت بحاجة إلى تعيين أدوار المسؤولين في Microsoft 365 [للأعمال](../add-users/assign-admin-roles.md) للمستخدمين الذين تضيفهم في المعالج، يمكنك القيام بذلك لاحقا في **صفحة** المستخدمون. 
  
إذا لم تكمل معالج الإعداد، يمكنك إكمال مهام الإعداد في أي وقت من [مركز](https://go.microsoft.com/fwlink/p/?linkid=2024339) >  **الإدارةالستعداد**. من هنا، يمكنك ترحيل البريد الإلكتروني وجهات الاتصال من خدمة بريد إلكتروني أخرى، وتغيير مجال حساب المسؤول، وإدارة معلومات الفوترة، وإضافة مستخدمين أو إزالتهم، وإعادة تعيين كلمات المرور، وإدارة وظائف أعمال أخرى. لمزيد من المعلومات حول الاختلافات بين معالج الإعداد وصفحة الإعداد،  راجع الاختلافات بين Microsoft 365 الإعداد [وصفحة الإعداد](o365-setup-wizard-and-setup-page.md).

إذا تعثرت في أي وقت، فاتصل بنا. [نحن هنا لمساعدتك!](../../business-video/get-help-support.md).
  
## <a name="when-not-to-use-the-setup-wizard-active-directory-synchronization-and-hybrid-environments"></a>عند عدم استخدام معالج الإعداد: مزامنة Active Directory والبيئات المختلطة

هناك بعض السيناريوهات التي تتضمن إما عملية نقل البيانات أو المستخدمين من البيئات المحلية أو إعداد نظام مختلط يتضمن مزامنة الدليل. إذا كنت في أي من الفئتين، فاتبع الإرشادات الواردة في المقالات التالية:
  
- لإعداد مزامنة الدليل مع Active Directory المحلية، راجع إعداد مزامنة الدليل ل [Microsoft 365](../../enterprise/set-up-directory-synchronization.md)، ولفهم نماذج الهوية المختلفة في Microsoft 365، اقرأ نشر البنية الأساسية للهوية [Microsoft 365.](../../enterprise/deploy-identity-solution-overview.md)

- لإعداد سجل Exchange مختلط، يمكن العثور على مجموعة كاملة من الإرشادات التي ترشدك عبر كل الطرق المختلفة لإعداد تبادل مختلط (بما في ذلك إعداد سجلات DNS) هنا: Exchange Server [مساعد](/exchange/exchange-deployment-assistant) النشر

- لإعداد موقع مختلط SharePoint، لا سيما ميزات البحث المختلط وميزات الموقع، راجع البحث المختلط [في](/SharePoint/hybrid/hybrid-search-in-sharepoint) SharePoint.

## <a name="move-to-microsoft-365-all-at-once-or-in-stages"></a>الانتقال إلى Microsoft 365 مرة واحدة أو على مراحل

- **هل تريد نقل مؤسستك إلى Microsoft 365 مرة واحدة؟** إذا كان الأمر كذلك، فخطط لنقل مجالك إلى Microsoft 365 مباشرة. ابدأ بتشغيل معالج Microsoft 365 الإعداد؛ وسيطالبك بإعداد مجالك.

- **هل تريد الانتقال إلى Microsoft 365 تدريجيا؟** إذا كنت تريد الانتقال إلى Microsoft 365 على مراحل، فتخطى تشغيل معالج إعداد Microsoft 365 وضع في اعتبارك Microsoft 365 الميزات بالترتيب التالي:

    1. [أضف الموظفين Microsoft 365](../add-users/add-users.md) بحيث يمكنهم تنزيل التطبيقات Office وتثبيتها.

    2. [قم بتنزيل تطبيقات Office وتثبيتها](https://support.microsoft.com/office/4414eaaf-0478-48be-9c42-23adc4716658) لاستخدام Word Excel والمحتوى PowerPoint على الكمبيوتر والأجهزة.

    3. [قم بإعداد Microsoft Teams](#plan-for-teams) لاستخدامها لاجتماعاتك.

    4. [نقل المحتوى إلى Microsoft 365 التخزين السحابي](set-up-file-storage-and-sharing.md) (OneDrive أو SharePoint الفريق).

    5. عندما تكون جاهزا، في مركز الإدارة[](https://go.microsoft.com/fwlink/p/?linkid=2024339)، حدد إعداد في  جزء التنقل الأيسر، ثم استخدم صفحة الإعداد [لنقل المجال والبريد الإلكتروني](add-domain.md).

## <a name="check-that-your-devices-meet-system-requirements"></a>التحقق من أن أجهزتك تلبي متطلبات النظام

يمكن لكل شخص في مؤسستك تثبيت مجموعة تطبيقات Office 2016 (Word و Excel و PowerPoint وغيرها) على ما يصل إلى خمسة أجهزة كمبيوتر شخصية و Mac. راجع متطلبات نظام التشغيل والكمبيوتر لتثبيت Office [2016](https://go.microsoft.com/fwlink/?LinkId=534827) للأعمال.
  
يمكن تثبيت تطبيقات الأجهزة المحمولة على أجهزة iOS وAndroid Windows الأجهزة. يمكنك العثور على معلومات حول دعم المستعرض والجهاز المحمول في [متطلبات النظام Office](https://go.microsoft.com/fwlink/?LinkId=534827).
  
## <a name="plan-for-email"></a>التخطيط للبريد الإلكتروني

إذا كنت تخطط للانتقال من خدمة بريد إلكتروني موجودة إلى Microsoft 365، يستغرق التبديل عادة يومين.
  
### <a name="plan-for-email-downtime"></a>التخطيط لفترة توقف البريد الإلكتروني
  
إذا كنت تريد استخدام Microsoft 365 البريد الإلكتروني:
  
- لنقل عنوان البريد الإلكتروني الخاص بأعمالك (*مثل rob\@ contoso.com*) من خدمة بريد إلكتروني أخرى إلى Microsoft 365، ستحتاج إلى توجيه البريد الإلكتروني لتسليمه إلى علبة بريدك Microsoft 365 البريد الجديدة. يمكنك القيام بذلك عن طريق تحديد  ترحيل بيانات المستخدمين على صفحة الإعداد، حيث نرشدك خلال التحديثات التي تحتاج إلى إجراءها في مضيف المجال، خطوة بخطوة.

- بعد تحديث مضيف المجال، يتم عادة إدخال التغييرات حيز التنفيذ في ساعة أو ساعتين فقط. ولكن يجب أن تدرك أن تحديث التغييرات عبر الإنترنت قد يستغرق أحيانا ما يصل إلى 72 ساعة.

- نظرا لأنه قد يكون لديك وقت توقف للبريد الإلكتروني، نوصيك بالتبديل إلى البريد الإلكتروني من Microsoft خلال المساء أو في نهاية الأسبوع عندما تتلقى رسائل بريد إلكتروني أقل.

### <a name="plan-to-move-your-existing-email-contacts-and-calendar"></a>التخطيط لنقل البريد الإلكتروني وجهات الاتصال والتقويم الحالي
  
إذا كنت تريد استخدام Microsoft 365 لحساب بريدك الإلكتروني، يمكنك إحضار البريد الإلكتروني وجهات الاتصال والتقويم الموجود معك. تساعدك **صفحة** الإعداد على نقل البريد الإلكتروني وجهات الاتصال الموجودة لمعظم السيناريوهات. لدينا أيضا أدلة خطوة بخطوة لنقل علبة بريد واحدة أو العديد من علب البريد.
  
|**ما عدد علب البريد؟**|**توصية**|
|:-----|:-----|
|عدد قليل فقط  <br/> |إذا كنت لا تريد استخدام صفحة الإعداد لترحيل علب البريد، يمكنك السماح لمالكي علب البريد بترحيل البريد الإلكتروني وجهات الاتصال الخاصة بهم. راجع [ترحيل البريد الإلكتروني وجهات الاتصال Microsoft 365 للأعمال](migrate-email-and-contacts-admin.md).  <br/> |
|العديد من  <br/> |إذا كنت تقوم بالترحيل من Gmail، فشاهد ترحيل علب بريد [G Suite Microsoft 365](/Exchange/mailbox-migration/migrating-imap-mailboxes/migrate-g-suite-mailboxes).  <br/> إذا كنت تقوم بالترحيل من موفر بريد إلكتروني آخر، بما في ذلك Exchange، فشاهد طرق ترحيل حسابات بريد إلكتروني متعددة [إلى](/Exchange/mailbox-migration/mailbox-migration) Microsoft 365.  <br/> |

## <a name="plan-for-file-storage-and-migration"></a>التخطيط لتخزين الملفات وا ترحيلها

Microsoft 365 التخزين السحابي للأفراد والمؤسسات الصغيرة والمؤسسات. للحصول على إرشادات حول ما يجب تخزينه، راجع أين يمكنك تخزين المستندات [في](https://support.microsoft.com/office/d18d21a0-1f9f-4f6c-ac45-d52afa0a4a2e) Microsoft 365.
  
- **يمكنك نقل مئات الملفات** إلى [OneDrive أو إلى](https://support.microsoft.com/office/45114744-6D42-45CD-8975-F9617819BDEB) موقع SharePoint [الفريق](https://support.microsoft.com/office/da549fb1-1fcb-4167-87d0-4693e93cb7a0#__toc384119242). يمكنك تحميل 100 ملف في كل مرة. تجنب تحميل الملفات التي يزيد حجمها عن 2 غيغابايت، وهو الحد الأقصى لحجم الملف بشكل افتراضي.
  
- **إذا كنت تريد نقل عدة آلاف** من الملفات Microsoft 365 التخزين، فراجع SharePoint [الإنترنت](/office365/servicedescriptions/sharepoint-online-service-description/sharepoint-online-limits). نوصي باستخدام أداة ترحيل أو التفكير في توظيف شريك لمساعدتك في عملية [](https://go.microsoft.com/fwlink/?linkid=391089) الترحيل. للحصول على معلومات حول كيفية ترحيل عدد كبير من الملفات، راجع SharePoint عبر الإنترنت [OneDrive دليل مستخدم الترحيل](/sharepointmigration/upload-on-premises-content-to-sharepoint-online-using-powershell-cmdlets).
  
## <a name="plan-for-teams"></a>التخطيط Teams

يمكنك استخدام Microsoft Teams إجراء مكالمات إلى أشخاص آخرين في مؤسستك ممن لديهم اشتراك. على سبيل المثال، إذا كانت مؤسستك تحتوي على 10 أشخاص، يمكنك الاتصال ببعضهم البعض والمراسلة الفورية باستخدام Teams دون أي إعداد خاص. لمزيد من المعلومات، راجع [بدء استخدام Microsoft Teams](/MicrosoftTeams/get-started-with-teams-quick-start).

بالنسبة إلى المؤسسات الكبيرة أو إذا كنت تبدأ من عمليات نشر Skype for Business أو عمليات نشر مختلط أو Skype for Business أو عمليات نشر مختلط، فشاهد كيفية نشر [Microsoft Teams.](/MicrosoftTeams/how-to-roll-out-teams)
  
## <a name="plan-for-integration-with-active-directory-or-other-software"></a>التخطيط للتكامل مع Active Directory أو برامج أخرى

- **هل تريد التكامل مع Active Directory الخاص بك في الموقع؟** يمكنك دمج Active Directory Microsoft 365 باستخدام Azure Active Directory الاتصال. للحصول على الإرشادات، راجع [إعداد مزامنة الدليل Microsoft 365](../../enterprise/set-up-directory-synchronization.md).
  
- **هل تريد دمج Microsoft 365 مع البرامج التي تقدمها شركات أخرى؟** إذا كنت بحاجة إلى Microsoft 365 مع برامج أخرى في مؤسستك، فإننا ننصحك بتوظيف شريك لمساعدتك في [](https://go.microsoft.com/fwlink/?linkid=391089) النشر.
  
## <a name="do-you-want-someone-to-help-you-set-up-microsoft-365"></a>هل تريد أن يساعدك أحد الأشخاص في إعداد Microsoft 365؟

- **إذا كان لديك أقل من 50 موظفا:**

  - **اطلب المساعدة وسنتصل بك**. بعد شراء Microsoft 365، يمكنك الوصول إلى مركز الإدارة (لست بحاجة إلى تشغيل الإعداد للوصول إليه). في أسفل مركز الإدارة، حدد **هل تحتاج إلى مساعدة؟** وصف مشكلتك، وسنتصل بك. 
  - **اتصل [Microsoft 365 دعم الأعمال](../../business-video/get-help-support.md) مع أسئلتك**. نحن هنا لمساعدتك! 
  - **فكر في توظيف [شريك Microsoft](https://go.microsoft.com/fwlink/?linkid=391089)**. إذا لم يكن لديك أي وقت، أو إذا كان لديك متطلبات متقدمة (مثل نقل الآلاف من الملفات إلى Microsoft 365 التخزين السحابي أو التكامل مع برامج أخرى)، يمكن أن يكون الشريك من ذوي الخبرة مساعدة كبيرة. 

- **إذا كان لديك أكثر من 50** [موظفا، FastTrack](https://go.microsoft.com/fwlink/?LinkId=517115) "مركز التوفر" لمساعدتك في عملية النشر.
