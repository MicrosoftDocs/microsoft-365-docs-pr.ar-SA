# <a name="update-your-admin-phone-number-and-email-address"></a>تحديث رقم هاتف المسؤول وعنوان البريد الإلكتروني

تشرح هذه المقالة كيف يمكنك أنت، المسؤول، تغيير عنوان بريدك الإلكتروني وهاتف العمل في Microsoft 365.
  
إذا كنت تبحث عن كيفية تغيير معلومات ملف التعريف الخاصة بالشركة، مثل اسم الشركة وعنوانها ورقم هاتف الشركة ومعلومات الاتصال التقنية، فاطلع على تغيير عنوان مؤسستك والبريد الإلكتروني للاتصال التقني ومعلومات [أخرى](change-address-contact-and-more.md).

لمزيد من المعلومات حول تغيير معلومات جهة اتصال المستخدم أو إزالة الموظفين السابقين، راجع [المحتوى ذي الصلة](#related-content).
  
## <a name="to-update-your-phone-number-and-email-address"></a>لتحديث رقم هاتفك وعنوان بريدك الإلكتروني

استخدم صفحة **معلومات الأمان** لتغيير رقم هاتفك الجوال وعنوان بريدك الإلكتروني البديل. يتم استخدام عنوان البريد الإلكتروني البديل والإعلامات المهمة، مثل إعادة تعيين كلمة مرور المسؤول (وليس كلمة مرور مسؤول الكمبيوتر). 
  
1. استعرض بحثا عن <a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">مركز مسؤولي Microsoft 365</a>.

2. في الرأس، حدد أيقونة ملف التعريف الخاص بك \> **معلومات أمان** \> **حسابي**.

3. في علامة **التبويب معلومات الأمان**، حدد **إضافة أسلوب** \>  \> الهاتف **بديل** الهاتف **البريد** الإلكتروني لإضافة التفاصيل. لتحديث تفاصيل عنوان البريد الإلكتروني البديل والهاتف والجوال، حدد **تغيير**. تأكد من استخدام شيء آخر غير عنوان بريدك الإلكتروني من Microsoft  عنوان بريدك الإلكتروني البديل.

    > [!IMPORTANT]
    > يجب استخدام عنوان البريد الإلكتروني البديل ورقم الهاتف الجوال لإعادة تعيين كلمة مرور المسؤول (وليس كلمة مرور مسؤول الكمبيوتر).

4. عند الانتهاء، حدد **حفظ**.
