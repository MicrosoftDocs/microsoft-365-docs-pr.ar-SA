# <a name="microsoft-365-usage-analytics-data-model"></a>Microsoft 365 بيانات تحليلات الاستخدام

## <a name="data-for-the-microsoft-365-usage-analytics-tables"></a>بيانات جداول Microsoft 365 تحليلات الاستخدام

Microsoft 365 تحليلات الاستخدام إلى API التي تعرض نموذج بيانات متعدد الأبعاد. إن واجهات برمجة التطبيقات Microsoft 365 التي تستخدمها تحليلات الاستخدام لإنشاء بياناتها هي من واجهات برمجة التطبيقات Graph المتوفرة بشكل عام. لا تتوفر بشكل عام Microsoft 365 API لتحليلات الاستخدام.
  
> [!NOTE]
> لمزيد من المعلومات، راجع استخدام [Microsoft 365 الاستخدام في Microsoft Graph](/graph/api/resources/report). 
  
توفر هذه API معلومات حول الاتجاه الشهري لاستخدام مختلف Microsoft 365 الخدمات. للحصول على البيانات الدقيقة التي تم إرجاعها بواسطة API، راجع الجدول في المقطع التالي.
