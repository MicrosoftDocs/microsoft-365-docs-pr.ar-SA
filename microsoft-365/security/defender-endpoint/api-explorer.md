# <a name="api-explorer"></a>مستكشف واجهة برمجة التطبيقات

[!INCLUDE [Microsoft 365 Defender rebranding](../../includes/microsoft-defender.md)]

**ينطبق على:**
- [Defender for Endpoint الخطة 2](https://go.microsoft.com/fwlink/p/?linkid=2154037)

إن Microsoft Defender لنقطة النهاية API Explorer هو أداة تساعدك على استكشاف واجهات برمجة تطبيقات Defender لنقطة النهاية بشكل تفاعلي.

يسهل مستكشف واجهة برمجة التطبيقات إنشاء استعلامات واجهة برمجة التطبيقات واختبارها وإرسالها لأي نقطة نهاية لواجهة برمجة تطبيقات Defender لنقطة النهاية. استخدم مستكشف واجهة برمجة التطبيقات لاتخاذ إجراءات أو البحث عن البيانات التي قد لا تكون متوفرة بعد من خلال واجهة المستخدم.

الأداة مفيدة أثناء تطوير التطبيق. يسمح لك بإجراء استعلامات واجهة برمجة التطبيقات التي تحترم إعدادات وصول المستخدم، ما يقلل من الحاجة إلى إنشاء رموز الوصول المميزة.

يمكنك أيضا استخدام الأداة لاستكشاف معرض استعلامات العينة، ونسخ نماذج التعليمات البرمجية للنتيجة، وإنشاء معلومات تتبع الأخطاء.

باستخدام مستكشف واجهة برمجة التطبيقات، يمكنك:

- تشغيل الطلبات لأي أسلوب ورؤية الاستجابات في الوقت الحقيقي.
- استعرض بسرعة من خلال نماذج واجهة برمجة التطبيقات وتعرف على المعلمات التي تدعمها.
- إجراء مكالمات واجهة برمجة التطبيقات بسهولة؛ لا حاجة للمصادقة خارج تسجيل الدخول إلى مدخل الإدارة.

## <a name="access-api-explorer"></a>Access API Explorer

من قائمة التنقل اليمنى، حدد **"Partners & APIs** \> **[API Explorer](https://security.microsoft.com/interoperability/api-explorer)**".

## <a name="supported-apis"></a>واجهات برمجة التطبيقات المدعومة

يدعم API Explorer جميع واجهات برمجة التطبيقات التي يقدمها Defender لنقطة النهاية.

تتوفر قائمة واجهات برمجة التطبيقات المدعومة في [وثائق واجهات برمجة التطبيقات](apis-intro.md).

## <a name="get-started-with-the-api-explorer"></a>بدء استخدام مستكشف واجهة برمجة التطبيقات

1. في الجزء الأيمن، توجد قائمة بنماذج الطلبات التي يمكنك استخدامها.
2. اتبع الارتباطات وانقر فوق **"تشغيل الاستعلام**".

قد تتطلب بعض العينات تحديد معلمة في عنوان URL، على سبيل المثال، {machine- ID}.

## <a name="faq"></a>الأسئلة المتداولة

**هل أحتاج إلى رمز مميز لواجهة برمجة التطبيقات لاستخدام مستكشف واجهة برمجة التطبيقات؟** <br>
بيانات الاعتماد للوصول إلى واجهة برمجة التطبيقات غير مطلوبة. يستخدم مستكشف واجهة برمجة التطبيقات الرمز المميز لمدخل إدارة Defender لنقطة النهاية كلما تقدم بطلب.

يتم استخدام بيانات اعتماد مصادقة المستخدم التي تم تسجيل دخولها للتحقق من أن مستكشف واجهة برمجة التطبيقات مخول بالوصول إلى البيانات نيابة عنك.

تكون طلبات واجهة برمجة التطبيقات المحددة محدودة استنادا إلى امتيازات RBAC. على سبيل المثال، يقتصر طلب "إرسال المؤشر" على دور مسؤول الأمان.
