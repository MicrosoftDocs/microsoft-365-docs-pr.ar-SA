## <a name="api-description"></a>وصف API
احصل على URI الذي يسمح بتنزيل حزمة ["التحقيق](collect-investigation-package.md)".

> [!IMPORTANT]
>
> - تتوفر هذه الإجراءات فقط للأجهزة التي تعمل Windows 10 والإصدار 1703 أو الإصدارات الأحدث وعلى Windows 11.

## <a name="limitations"></a>القيود

قيود السعر ل API هذه هي 2 مكالمة في الدقيقة و120 مكالمة في الساعة. 

## <a name="permissions"></a>الأذونات

أحد الأذونات التالية مطلوبة لاستدعاء API هذه. لمعرفة المزيد، بما في ذلك كيفية اختيار الأذونات، راجع [الوصول إلى Microsoft Defender ل واجهات برمجة تطبيقات نقطة النهاية](apis-intro.md)

نوع الإذن|الإذن|اسم عرض الأذونات
:---|:---|:---
Application|Machine.Read.All|'قراءة كل ملفات تعريف الجهاز'
Application|"Machine.ReadWrite.All|"قراءة كل معلومات الجهاز وكتابتها"
مفوض (حساب العمل أو المدرسة)|Machine.CollectForensics|'تجميع الأطباء الشرعيين'

> [!NOTE]
> عند الحصول على رمز مميز باستخدام بيانات اعتماد المستخدم:
>
> - يجب أن يكون لدى المستخدم إذن الدور التالي على الأقل: "تنبيهات التحقيق" (راجع [إنشاء أدوار وإدارتها](user-roles.md) للحصول على مزيد من المعلومات)
> - يحتاج المستخدم إلى الوصول إلى الجهاز، استنادا إلى إعدادات مجموعة الأجهزة (راجع [إنشاء مجموعات](machine-groups.md) الأجهزة وإدارتها للحصول على مزيد من المعلومات)

## <a name="http-request"></a>طلب HTTP

```http
GET https://api.securitycenter.microsoft.com/api/machineactions/{machine action id}/getPackageUri
```

## <a name="request-headers"></a>طلب رؤوس

الاسم|النوع|الوصف
:---|:---|:---
التخويل|سلسلة|حامل {token}. **مطلوب**.

## <a name="request-body"></a>طلب الحصول على "هيئة"

فارغ

## <a name="response"></a>الاستجابة

إذا نجح هذا الأسلوب، فيرجع رمز الاستجابة 200، موافق مع كائن يحتفظ بالارتباط إلى الحزمة في المعلمة "value". هذا الارتباط صالح لفترة قصيرة جدا ويجب استخدامه على الفور لتنزيل الحزمة إلى مساحة تخزين محلية. إذا كان إجراء الجهاز للمجموعة موجودا ولكنه غير مكتمل، فإن ذلك يرجع 404 لم يتم العثور عليه.
