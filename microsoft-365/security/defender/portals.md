# <a name="microsoft-security-portals-and-admin-centers"></a>مداخل الأمان ومراكز الإدارة في Microsoft

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]

على [Microsoft 365 Defender المدخل](microsoft-365-defender.md#the-microsoft-365-defender-portal) هو المنزل الجديد لمراقبة الأمان وإدارته عبر الهويات والبيانات والأجهزة والتطبيقات، ستحتاج إلى الوصول إلى مداخل مختلفة لمهام معينة متخصصة.

> [!TIP] 
> للوصول إلى مختلف المداخل ذات الصلة من Microsoft 365 Defender، حدد **موارد إضافية** في جزء التنقل.

## <a name="security-portals"></a>مداخل الأمان

يمكن لمشغلي الأمان لمسؤوليه الانتقال إلى المداخل التالية لإدارة الإعدادات الخاصة ب الأمان، التحقيق في أنشطة التهديدات المحتملة، والاستجابة للتهديدات النشطة، والتعاون مع المسؤولين في مجال المعلومات في حل المشاكل.
<p></p>

| اسم المدخل | الوصف | ارتباط |
|---|---|---| 
| Microsoft 365 Defender المدخل | مراقبة نشاط التهديدات والاستجابة له وتعزيز وضع الأمان عبر الهويات والبريد الإلكتروني والبيانات ونقاط النهاية والتطبيقات باستخدام Microsoft 365 Defender [](microsoft-365-defender.md) | [security.microsoft.com](https://security.microsoft.com/) |
| مركز حماية Microsoft Defender | مراقبة نشاط التهديدات على نقاط النهاية والاستجابة له باستخدام الإمكانات المتوفرة مع [Microsoft Defender لنقطة النهاية](/windows/security/threat-protection/microsoft-defender-atp/microsoft-defender-advanced-threat-protection) | [securitycenter.windows.com](https://securitycenter.microsoft.com/) |
| مركز & الأمان | إدارة [Exchange Online Protection](../office-365-security/exchange-online-protection-overview.md) [و Microsoft Defender Office 365](/microsoft-365/security/office-365-security/defender-for-office-365) لحماية البريد الإلكتروني وخدمات التعاون، وضمان الامتثال لأنظمة معالجة البيانات المختلفة | [protection.office.com](https://protection.office.com) |
| مدخل Defender for Cloud | استخدام [Microsoft Defender ل السحابة](/azure/security-center/security-center-intro) لتعزيز وضعية الأمان لمراكز البيانات وأحمال العمل المختلطة في السحابة | [portal.azure.com/#blade/Microsoft_Azure_Security](https://portal.azure.com/#blade/Microsoft_Azure_Security/SecurityMenuBlade/0) |
| مدخل Microsoft Defender for Identity | تحديد التهديدات المتقدمة والهويات المتهكة والإجراءات الضارة ل insider والكشف عنها التحقيق فيها باستخدام إشارات Active Directory باستخدام [Microsoft Defender for Identity](/azure-advanced-threat-protection/what-is-atp) | [portal.atp.azure.com](https://portal.atp.azure.com/) |
| مدخل Defender for Cloud Apps | استخدام [Microsoft Defender لتطبيقات السحابة](/cloud-app-security/what-is-cloud-app-security) للحصول على رؤية غنية والتحكم في سفر البيانات وتحليلات متطورة لتحديد مكافحة الهجمات الإلكترونية على الخدمات السحابية | [portal.cloudappsecurity.com](https://portal.cloudappsecurity.com/) |
| التحليل الذكي لمخاطر الأمان من Microsoft المدخل | احصل على تحديثات معلومات الأمان ل Microsoft Defender لنقطة النهاية، وأرسل عينات، واستكشف الموسوعة التي تشكل خطرا | [microsoft.com/wdsi](https://microsoft.com/wdsi) |

## <a name="portals-for-other-workloads"></a>مداخل لأحمال العمل الأخرى

على الرغم من أن هذه المداخل ليست مخصصة لإدارة الأمان بشكل خاص، فإنها تدعم أحمال العمل والمهام المختلفة التي يمكن أن تؤثر على أمانك. تفضل بزيارة هذه المداخل لإدارة الهويات والأذونات وإعدادات الأجهزة ونهج معالجة البيانات.
<p></p>

| اسم المدخل | الوصف | ارتباط | 
|---|---|---| 
| مدخل Azure | عرض جميع موارد [Azure وإدارتها](/azure/azure-resource-manager/management/overview)  | [portal.azure.com](https://portal.azure.com/) |
| مدخل Azure Active Directory | عرض [Azure Active Directory](/azure/active-directory/fundamentals/active-directory-whatis) وإدارته | [aad.portal.azure.com](https://aad.portal.azure.com/) |
| مركز التوافق في Microsoft 365 | إدارة سياسات معالجة البيانات وضمان [التوافق مع اللوائح](/compliance/regulatory/offering-home) | [compliance.microsoft.com](https://compliance.microsoft.com/) |
| مركز مسؤولي Microsoft 365 | تكوين Microsoft 365 وإدارة الأدوار والتراخيص وتعقب التحديثات لخدماتك Microsoft 365 | [admin.microsoft.com](https://go.microsoft.com/fwlink/p/?linkid=2166757) |
| إدارة نقاط النهاية من Microsoft إدارة | استخدام [إدارة نقاط النهاية من Microsoft](/mem/configmgr/) لإدارة الأجهزة وتأمينها باستخدام قدرات إدارة التكوين و Intune المدمجة | [devicemanagement.microsoft.com](https://devicemanagement.microsoft.com/) |
| Microsoft Intune المدخل | استخدام [Microsoft Intune](/intune/fundamentals/what-is-intune) لنشر سياسات الأجهزة ومراقبة الأجهزة للتوافق | [endpoint.microsoft.com](https://endpoint.microsoft.com/#blade/Microsoft_Intune_DeviceSettings/DevicesMenu/overview)
