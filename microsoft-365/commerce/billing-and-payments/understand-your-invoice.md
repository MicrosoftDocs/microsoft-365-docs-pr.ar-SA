# <a name="understand-your-bill-or-invoice"></a>فهم الفاتورة أو الفاتورة

توفر الفاتورة ملخصا لمصاريفك وإرشاداتك للدفع. يمكنك عرض [الفاتورة عبر الإنترنت](#view-your-online-invoice) في مركز مسؤولي Microsoft 365. يمكنك أيضا تنزيله في Portable Document Format (.pdf) لإرساله عبر البريد الإلكتروني.

لعرض الفاتورة وطباعتها:

1. في صفحة **الفوترة** >  <a href="https://go.microsoft.com/fwlink/p/?linkid=2102895" target="_blank">&</a> الدفعات، حدد نطاق تاريخ الفاتورة.
2. لطباعة نسخة PDF من الفاتورة أو حفظها، حدد **تنزيل الفاتورة PDF**، ثم اطبع ملف PDF.

لمعرفة المزيد، راجع [عرض الفاتورة أو الفاتورة](view-your-bill-or-invoice.md).

إذا كان لديك اشتراك Microsoft 365 فقط، فشاهد فهم الفاتورة [أو الفاتورة Microsoft 365 للأعمال](understand-your-invoice2.md).

## <a name="understand-the-invoice-header"></a>فهم رأس الفاتورة

تحدد أعلى الصفحة الأولى من هو الشخص الذي يمكن دفعه، وأين يتم إرسال الفاتورة إليه، وملخص الرسوم.

| المدة | الوصف |
| --- | --- |
| مباع إلى |حساب الفوترة الذي يحدد اسم وعنوان الكيان القانوني المسؤول عن الدفع. يمكن إدارة هذه المعلومات على صفحة حسابات <a href="https://go.microsoft.com/fwlink/p/?linkid=2084771" target="_blank"></a> الفوترة، حيث يمكنك العثور على اتفاقية الحساب وإدارة الأدوار والأذونات. |
| فاتورة إلى |تحديد من يتلقى الفاتورة. يمكن إدارة هذه المعلومات على <a href="https://go.microsoft.com/fwlink/p/?linkid=2103629" target="_blank">صفحة ملفات تعريف الفوترة</a> . يظهر ملف تعريف الفوترة أيضا على صفحة الفاتورة عبر الإنترنت، في المقطع **ملخص الفاتورة** . لمعرفة المزيد حول ملفات تعريف الفوترة وكيفية استخدامها لإنشاء خيارات فوترة أكثر مرونة لمنظمتك، راجع [إدارة ملفات تعريف الفوترة](manage-billing-profiles.md). |
| ملف تعريف الفوترة |اسم ملف تعريف الفوترة المستخدم لتعريف خصائص الفاتورة مثل **الفاتورة إلى**، **رقم أمر** الشراء، شروط الدفع. يمكن إدارة هذه المعلومات على <a href="https://go.microsoft.com/fwlink/p/?linkid=2103629" target="_blank">صفحة ملفات تعريف الفوترة</a> . لمزيد من المعلومات حول ملفات تعريف الفوترة وكيفية استخدامها لإنشاء خيارات فوترة أكثر مرونة لمنظمتك، راجع [إدارة ملفات تعريف الفوترة](manage-billing-profiles.md). |
| رقم الفاتورة |رقم فاتورة فريد تم إنشاؤه من Microsoft يستخدم لأغراض التعقب. |
| تاريخ الفاتورة |تاريخ إنشاء الفاتورة، عادة من خمسة إلى 12 يوما بعد نهاية دورة الفوترة. يمكنك التحقق من تاريخ الفاتورة على صفحة تفاصيل ملف تعريف الفوترة. يتم تضمين الرسوم التي تحدث بين نهاية فترة الفوترة وتاريخ الفاتورة في الفاتورة للشهر التالي، نظرا لأنها في فترة الفوترة التالية. يتم سرد تواريخ بدء وفترة الفوترة لكل فاتورة في الفاتورة PDF أعلاه **"ملخص الفوترة**".|
| شروط الدفع |كيفية الدفع مقابل فاتورة Microsoft. *صافي 30 يوما* يعني أنك تدفع باتباع الإرشادات الواردة في الفاتورة، في غضون 30 يوما من تاريخ الفاتورة. |

## <a name="understand-the-billing-summary"></a>فهم ملخص الفوترة

يعرض **ملخص الفوترة** ملخص الرسوم منذ فترة الفوترة السابقة وأي اعتمادات تم تطبيقها وضريبة وإجمالي المبلغ المستحق.

| المدة | الوصف |
| --- | --- |
| الرسوم|إجمالي عدد المنتجات التي تم شراؤها لفترة الفوترة هذه ورسومها وضرائبها ذات الصلة. يتم تجميع عمليات الشراء لتوفير طريقة عرض موجزة ل الفاتورة. |
| الائتمانات |الرصيد الذي تلقيته من الإرجاع |
| تم تطبيق أرصدة Azure |تقوم أرصدة Azure التي يتم تطبيقها تلقائيا على Azure بمصاريف كل فترة فوترة. إذا لم يكن لديك أي رصيد Azure، فهذا الحقل مخفي. لمزيد من المعلومات حول أرصدة Azure، راجع [تعقب رصيد Azure](/azure/billing/billing-mca-check-azure-credits-balance) اتفاقيات عملاء Microsoft. |
| الاكتتال الفرعي |المبلغ المستحق قبل الضريبة |
| الضريبة |نوع الضريبة التي تدفعها ومبلغها، استنادا إلى بلد ملف تعريف الفوترة. إذا لم تكن بحاجة إلى دفع الضريبة، فلا يتم عرض أي ضريبة على الفاتورة. |

### <a name="understand-your-charges"></a>فهم الرسوم

تظهر صفحات الرسوم التكلفة مقطوعة حسب المنتج. بالنسبة لعملاء Azure، قد يتم تنظيم الرسوم حسب قسم الفاتورة. لمزيد من المعلومات حول كيفية استخدام مقاطع الفواتير مع منتجات Azure، راجع مقاطع الفواتير [](/azure/billing/billing-mca-overview#invoice-sections) في بدء استخدام حساب الفوترة الخاص باتفاقية [عملاء Microsoft](/azure/billing/billing-mca-overview). ضمن كل طلب منتج، يتم تقسيم التكلفة حسب عائلة الخدمات.

| المدة |الوصف |
| --- | --- |
| سعر الوحدة | سعر الوحدة الفعال للخدمة (بعملة الأسعار) المستخدمة لحساب الرسوم. هذا السعر فريد للمنتج وعائلة الخدمة والعداد والعرض. |
| Qty | الكمية التي تم شراؤها أو استهلاكها خلال فترة الفوترة |
| الرسوم/الائتمانات | المبلغ الصافي للمصاريف بعد تطبيق الائتمانات/المبالغ المستردة |
| رصيد Azure | مبلغ أرصدة Azure المطبقة على الرسوم/الائتمانات |
| معدل الضريبة | معدل الضريبة، استنادا إلى البلد |
| مبلغ الضريبة | مقدار الضريبة المطبقة على عملية الشراء استنادا إلى معدل الضريبة |
| الإجمالي | المبلغ الإجمالي المستحق للشراء |

تختلف تفاصيل عناصر السطر استنادا إلى نوع المنتج الذي يتم محاسبتك عليه. على سبيل المثال، بالنسبة لمنتجات Azure، يتم عرض مقدار أرصدة Azure المطبقة. تظهر المنتجات المستندة إلى المقاعد سعر الوحدة وكميتها. تظهر تفاصيل الفاتورة المنتجات التي تم شراؤها والخصم أو الائتمانات التي تم تطبيقها ومعدل الضريبة والمبلغ وإجماليات عناصر السطر.

> الإجمالي = الرسوم - Azure Credit + Tax

يتم حساب المبلغ الإجمالي المستحق لكل عائلة خدمات من خلال طرح أرصدة Azure من الرصيد/الرسوم وإضافة الضريبة:

> الإجمالي = الرسوم/الائتمانات - Azure Credit + Tax

إذا كانت هناك رسوم Azure على فاتورتك ترغب في الحصول على مزيد من التفاصيل حولها، فراجع [مراجعة فاتورة اتفاقية عملاء Microsoft](/azure/cost-management-billing/understand/review-customer-agreement-bill).

## <a name="understand-the-last-invoice-page"></a>فهم صفحة الفاتورة الأخيرة

### <a name="payment-instructions"></a>إرشادات الدفع

في أسفل الفاتورة، توجد إرشادات حول كيفية دفع الفاتورة. يمكنك الدفع بواسطة سلكي أو شيك أو عبر الإنترنت.

### <a name="publisher-information"></a>Publisher المعلومات

إذا كانت لديك خدمات جهة خارجية في فاتورتك، يتم إدراج اسم وعنوان كل ناشر في أسفل الفاتورة.

## <a name="view-your-online-invoice"></a>عرض الفاتورة عبر الإنترنت

تتوفر الفواتير عبر الإنترنت. يتوفر ارتباط إلى فاتورتك عبر الإنترنت من فاتورة PDF ومن إعلام بالبريد الإلكتروني. يمكن توسيع الفاتورة عبر الإنترنت بحيث يمكنك عرض الرسوم على فاتورتك لمشاهدة المزيد من التفاصيل لكل عنصر. تتضمن الفاتورة عبر الإنترنت:

- **تفاصيل الأسعار**&mdash; معلومات إضافية بما في ذلك تفاصيل حول الخصومات وأسعار المنتجات.
- **الدفع عبر الإنترنت**&mdash; يمكنك اختيار إجراء عملية دفع عبر الإنترنت من الفاتورة.
- **إدارة تكلفة**&mdash; Azure بالنسبة لعملاء Azure، تتضمن الفواتير عبر الإنترنت ارتباط إلى إدارة تكلفة Azure.

### <a name="to-view-your-online-invoice"></a>لعرض فاتورتك عبر الإنترنت

1. في مركز الإدارة، انتقل إلى **صفحة فواتير** \> الفوترة & <a href="https://go.microsoft.com/fwlink/p/?linkid=2102895" target="_blank">الدفعات</a> .
2. لتنزيل إصدار .pdf من الفاتورة، اختر تنزيل الفاتورة **PDF** في الصف الخاص بالفاتورة التي تريد رؤياها.
3. لعرض فاتورتك عبر الإنترنت، اختر فاتورة من القائمة. يمكنك أيضا تنزيل .pdf من صفحة تفاصيل الفاتورة.

## <a name="invoice-faq"></a>الأسئلة الشائعة حول الفاتورة

### <a name="when-is-my-invoice-available"></a>متى تتوفر فاتورتي؟

يتم إنشاء بعض الفواتير في غضون 24 ساعة من عملية الشراء. يتم إنشاء فواتير أخرى في نهاية فترة الفوترة وهي تتضمن كل العناصر من تلك الفترة.

### <a name="how-do-i-pay-the-amount-due-on-my-invoice"></a>كيف يمكنني دفع المبلغ المستحق على الفاتورة؟

تعتمد إرشادات الدفع على طريقة الدفع الخاصة بك، كما يتم توفيرها في أسفل الفاتورة PDF. إذا كانت طريقة الدفع الخاصة بك هي بطاقة ائتمان، يتم شحنها تلقائيا في غضون 10 أيام من تاريخ الفاتورة. إذا كانت طريقة الدفع الخاصة بك عن طريق التحقق أو التحويل البنكي، فشاهد المعلومات الموجودة ضمن **إرشادات الدفع** في ملف PDF.

### <a name="whats-the-difference-between-sold-to-and-bill-to-addresses"></a>ما الفرق بين عناوين "مباع إلى" و"فاتورة إلى"؟

- **مباع إلى:** الكيان القانوني المسؤول عن الدفع والمعرف في الفاتورة. يتم استخدام العنوان الموفر هنا لتحديد معدل الضريبة ما لم تختار توفير عنوان شحن بديل أثناء عملية الشراء. لمزيد من المعلومات، راجع [معلومات الضرائب](tax-information.md).
- **الفاتورة إلى:** العنوان الذي يتم إرسال الفاتورة الفعلية فيه، إذا كان ذلك قابلا للتطبيق. يمكن أن يكون هناك عدة **فواتير عناوين** لكل كيان قانوني، ولكن فاتورة **واحدة** فقط يجب معالجتها لكل ملف تعريف فوترة.

### <a name="what-are-billed-amount-and-amount-due"></a>ما هي "المبلغ الذي تم فوترته" و"المبلغ المستحق؟"

- **المبلغ المفوتر:** المبلغ الإجمالي للشراء الذي قمت به.
- **المبلغ المستحق:** الرصيد المتبقي لما تدين به.

### <a name="what-is-the-difference-between-service-period-and-billing-period"></a>ما الفرق بين "فترة الخدمة" و"فترة الفوترة"؟

- **فترة الخدمة:** الفترة الزمنية التي يتم خلالها محاسبتك على استخدام الخدمة.
- **فترة الفوترة:** الفترة الزمنية منذ تاريخ الفاتورة الأخيرة.

### <a name="why-dont-i-see-azure-prepayment-as-a-payment-method"></a>لماذا لا أرى دفعة Azure المسبقة كطريقة دفع؟

تتوفر دفعة Azure المسبقة كطريقة دفع فقط لمنتج وخدمات Azure المؤهلة

## <a name="need-help-contact-support"></a>هل تحتاج إلى مساعدة؟ الاتصال بالدعم

إذا كانت لديك أسئلة أو كنت بحاجة إلى مساعدة في استخدام أرصدة Azure، <a href="https://portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade/newsupportrequest" target="_blank">فنشئ طلب دعم باستخدام دعم Azure</a>.

إذا كانت لديك أسئلة أو كنت بحاجة إلى مساعدة في الفاتورة مركز مسؤولي Microsoft 365، فاتصل ب [دعم منتجات الأعمال](../../admin/get-help-support.md).
