# <a name="microsoft-cloud-for-it-architects-illustrations"></a>رسومات توضيحية لمهندسي تكنولوجيا المعلومات في سحابة Microsoft

توفر لك ملصقات هندسة السحابة هذه معلومات حول خدمات Microsoft السحابية، بما في ذلك Microsoft 365 وAzure Active Directory (Azure AD) وMicrosoft Intune وMicrosoft Dynamics 365 والحلول المختلطة المحلية والسحابية. 

يمكن لصانعي القرار والمهندسين المعماريين في تكنولوجيا المعلومات استخدام هذه الموارد لتحديد الحلول المثالية لأحمال العمل الخاصة بهم واتخاذ قرارات بشأن مكونات البنية الأساسية مثل الشبكات والهوية والأمان.

<a name="zero trust"></a>
### <a name="microsoft-365-zero-trust-deployment-plan"></a>خطة نشر الثقة المعدومة من Microsoft 365


يوفر هذا الرسم التوضيحي خطة نشر لبناء أمان الثقة المعدوم مع Microsoft 365. الثقة المعدومة هي نموذج أمان جديد يفترض الخرق ويتحقق من كل طلب كما لو كان ينشأ من شبكة غير خاضعة للرقابة. بغض النظر عن مكان إنشاء الطلب أو المورد الذي يصل إليه، يعلمنا نموذج الثقة المعدومة "عدم الثقة أبدا، التحقق دائما".
